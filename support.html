TallyCut Support

Help, bug reports, and feature requests for TallyCut.

If you need help with TallyCut, please open an issue in the support repository.

Open a Support Issue

What to include

  • Your iPhone model and iOS version.
  • The TallyCut app version.
  • A short description of what you expected to happen.
  • What actually happened, including any error message shown by the app.
  • Steps that reproduce the issue, if possible.

Privacy reminder

GitHub Issues are public. Please do not post private videos, personal information, payment information, or sensitive project files in a public issue.

Subscriptions and billing

TallyCut subscriptions are handled by Apple through the App Store. For billing, cancellation, or refund requests, please use Apple’s subscription and purchase support.

支持

TallyCut 的帮助、问题反馈和功能建议。

如果你在使用 TallyCut 时遇到问题，请在支持仓库提交 Issue。

提交支持 Issue

建议提供的信息

  • 你的 iPhone 型号和 iOS 版本。
  • TallyCut App 版本。
  • 你原本期望发生什么。
  • 实际发生了什么，包括 App 显示的错误信息。
  • 如果可以，请提供复现步骤。

隐私提醒

GitHub Issues 是公开页面。请不要在公开 Issue 中发布私人视频、个人信息、付款信息或敏感工程文件。

订阅和账单

TallyCut 的订阅由 Apple 通过 App Store 处理。账单、取消订阅或退款请求，请使用 Apple 的订阅和购买支持渠道。
